# Vue-Typescript

### 简介
一款基于Typescript语言开发的Vue框架模板，使用webpack4构建，样式使用scss模块化开发，主要适配移动端的一款H5应用

### 主要技术
* Typescript3
* Webpack4
* Vue
* Vue-router
* Sass
* axios

### 使用

开发模式:

```sh
git clone git@github.com:xpioneer/vue-typescript.git

cd vue-typescript

yarn

yarn start
```

### 生产构建

```sh
yarn build
```
